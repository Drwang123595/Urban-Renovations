# 提示词深度优化方案：基于用户新指标体系

收到您新的指标定义，这比之前的版本更加精细，特别是**空间等级**的划分发生了重大变化。我将根据您的最新指示，对提示词进行全面重构。

## 1. 指标体系更新

### 1.1 城市更新 (Urban Renewal) - 需“认真思考”
*   **核心痛点**: 之前的定义可能过于宽泛或死板。
*   **新策略**: 引入**“核心意图判定” (Core Intent Detection)**。
    *   不仅仅看关键词，要看研究的**目的**是否是为了解决存量空间的问题（衰退、低效、不公平等）。
    *   **判定逻辑**:
        *   IF (Target = Existing Built Environment) AND (Action = Improve/Regenerate/Revitalize) -> **Yes**.
        *   IF (Target = New Land) OR (Action = Expansion) -> **No**.

### 1.2 空间研究 (Spatial Study) - “针对具体空间的具体事务”
*   **核心痛点**: 排除“非研究空间的干扰”（如背景介绍、举例提及）。
*   **新定义**: **Research Subject Alignment** (研究对象对齐)。
    *   只有当“空间”是“研究的具体事务”的**载体**时，才算空间研究。
    *   例如：“以北京为例分析房价” -> **Yes**（北京是房价的载体）。
    *   “中国正处于快速城市化阶段，本文提出了一个模型...” -> **No**（中国只是背景）。

### 1.3 空间等级 (Spatial Level) - **全新 9 级分类**
旧分类已被废弃，需完全替换为以下 9 个层级：
1.  **Global** (全球)
2.  **Multi-country/Region** (多国/多地区) - *注：这里的 Region 指跨国区域*
3.  **Country/Region** (全国/地区) - *注：这里的 Region 指国家级地区，如“中国香港”或“美国西部”*
4.  **Multi-province/State** (多省/州)
5.  **Province/State** (全省/全州)
6.  **Urban Agglomeration** (城市群)
7.  **Single City** (单城市) - *含全市范围或模糊的城市层级*
8.  **Multi-neighborhood/Block** (多街区) - *跨街区、多社区对比*
9.  **Single Neighborhood/Block** (单街区) - *微观尺度，如单个小区、单条街道*

### 1.4 具体空间描述 (Specific Description)
*   **要求**: 必须是**研究对象**的空间，严格剔除背景空间。
